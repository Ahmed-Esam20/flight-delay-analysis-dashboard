#  Flight Delay Analysis Dashboard (Power BI)

## Overview
This project analyzes flight delays to uncover patterns in airline performance, airport efficiency, and delay causes.

## Key Insights
- Late Aircraft Delay is the main contributor (~40%)
- Some airports have high delays despite low traffic
- Delay patterns vary by hour and day

## Tools Used
- Power BI
- DAX
- Power Query
